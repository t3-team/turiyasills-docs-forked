# Features Overview

Turiyaskills offers two powerful applications designed to revolutionize talent acquisition and evaluation. The **Assessment App** provides comprehensive skills testing with **SMARTEVAL** (quiz-based assessments) and **ASSESS360** (survey-based assessments), enabling precise candidate evaluation through AI-powered questions and secure testing environments. The **Top1 App** focuses on recruitment automation, featuring advanced job description parsing, resume analysis, candidate scoring, and AI-driven insights to help you recruit the top 1% cream of applications through thorough screening and filtering. Together, these applications create an end-to-end solution that streamlines the entire hiring process from candidate sourcing to final evaluation.

## 🚀 Core Assessment Platform

### SMARTEVAL: Quiz-Based Assessments
- **Multi-Section Design**: Create structured assessments with multiple skill sections
- **Advanced Question Types**: Support for coding, multiple choice, short text, and image-based questions
- **AI Question Generation**: Automatically generate relevant questions based on job descriptions
- **Secure Testing**: Safe Exam Browser (SEB) integration and video proctoring capabilities
- **Question Randomization**: Prevent cheating with randomized question and answer orders
- **Bulk Import**: Upload questions via Excel/JSON templates or import from question banks

### ASSESS360: Survey-Based Assessments  
- **Multi-Section Surveys**: Build comprehensive behavioral and skills surveys
- **Branching Logic**: Dynamic question flow based on previous responses
- **AI-Powered Reports**: Generate personalized feedback and insights automatically
- **Tag-Based Scoring**: Implement sophisticated scoring rules and analytics
- **Google Sheets Integration**: Real-time data synchronization and tracking
- **Time Constraints**: Set time limits for realistic assessment scenarios
