# Features Overview

- Turiyaskills offers a comprehensive assessment platform with **SMARTEVAL** and **ASSESS360**, providing AI-driven evaluations for both technical and behavioral competencies. 
- **SMARTEVAL** enables quiz-based assessments with automated scoring, detailed analytics, and customizable settings to evaluate skills effectively. 
- **ASSESS360** focuses on survey-based assessments, offering personalized AI reports, multi-section surveys, and branching logic to assess behavioral attributes. 

The platform also includes features such as performance insights, completion tracking, and personalized learning recommendations to help users identify strengths and areas for improvement.

## Key Features

### 1. AI-Powered Skill Assessment
Harness the power of artificial intelligence to streamline skill evaluations. Turiyaskills ensures precise, data-driven insights to help you identify top talent effortlessly.

### 2. Customizable Assessments
- Tailor assessments for diverse domains, including software development, electronic engineering, and more.
- Create quizzes with various question types, including multiple-choice, coding challenges, and subjective answers.

### 3. Advanced Security Features
- Safe Exam Browser Integration: Secure your assessments by preventing unauthorized actions and ensuring focus.

### 4. Flexible Plans
- Free Plan: Start with 100 AI credits and 20 response credits for a trial experience.
- Pro Plan: Unlock advanced features like AI-powered reporting and video proctoring for enhanced functionality.

### 5. Effortless Credit Management
- Manage your usage with AI and response credits tailored to your needs.
- Use the Response Credit Calculator to plan and optimize your assessments.

### 6. Detailed Analytics and Reporting
- Gain actionable insights with real-time performance analytics.
- Download customizable reports in Excel or PDF formats.
- AI-powered feedback ensures transparency and clarity in evaluations.

### 7. User-Friendly Experience
- Choose between Chat Mode (sequential question navigation) or Standard Mode (all questions on one page).
- Easy-to-use interfaces for both test creators and participants.

### 8. Workflow Automation (Coming Soon)
- Automate recruitment pipelines with AI-powered resume parsing and candidate screening.
- Configure stages like first-round assessments, coding tests, and more for a seamless hiring process.

### 9. In-Development Features
- Circuit & Coding Simulation: Offers a proprietary circuit simulation engine tailored for evaluating practical skills in hardware and electronics engineering roles, along with coding simulation integrated with GitHub Codespaces
- Enhanced Recruitment Analytics: Track metrics like time-to-hire and source effectiveness.
- The analytics features under development include individual and cumulative assessment analysis. These features provide detailed skill breakdowns across subjects offering performance insights. They also include question analysis by difficulty and correctness, tracking of time spent per section, and completion rates. 
- Additionally, personalized learning recommendations will be available, helping users identify their strengths, weaknesses, and areas for improvement based on assessment results.
