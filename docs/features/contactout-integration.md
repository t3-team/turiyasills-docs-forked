# How to use ContactOut API Key in TuriyaSkills Top1 App

## Overview
ContactOut integration allows you to search millions of candidate profiles, import matching resumes, and score them through your recruitment pipeline stages for efficient hiring decisions.

## Step 1: Get a valid API key from ContactOut
Reach out to https://contactout.com/meeting to schedule a meeting and obtain your ContactOut API key. This key enables access to their candidate database.

**[Schedule ContactOut Meeting](https://contactout.com/meeting)**

## Step 2: Key in license key and validate in Settings → Integrations (marketplace)
Navigate to the Settings page, go to the Integrations section (marketplace), enter your ContactOut API key, and click "Save & Validate" to verify the key's authenticity.

**Path:** Settings → Integrations → ContactOut People Search License → Enter API Key → Save & Validate

## Step 3: Start using ContactOut in the resume pipeline connectors
Once validated, return to Resume Parser → expand "Resume Pipeline Connectors" → use ContactOut People Search to search matching resumes for the uploaded job and score them to proceed recruitment pipeline stages.

### Features Available:
- Search matching resumes based on job requirements
- Smart skill matching with percentage scoring
- Import selected candidates for resume scoring
- Proceed candidates through recruitment pipeline stages

## ✅ Benefits of ContactOut Integration

• **Access to millions of verified candidate profiles** - Expand your talent pool beyond traditional sources

• **Automated skill matching and candidate ranking** - AI-powered matching saves time and improves quality

• **Direct email and contact information access** - Reach out to candidates immediately

• **Seamless integration with Top1 scoring pipeline** - Streamlined workflow from search to hire

• **Bulk candidate selection and import capabilities** - Process multiple candidates efficiently

## Getting Started

1. **Prerequisites**: Ensure you have a valid ContactOut subscription and API access
2. **Setup**: Complete the integration steps outlined above
3. **Usage**: Start searching and importing candidates through the Resume Pipeline Connectors
4. **Support**: Contact support if you encounter any issues during setup or usage

For additional help with ContactOut integration, please reach out to our support team.
