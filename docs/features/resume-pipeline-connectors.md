# Resume Pipeline Connectors

Resume Pipeline Connectors provide powerful tools to enhance your recruitment process by connecting to external databases and leveraging your existing candidate data. These connectors streamline candidate sourcing, matching, and import processes.

## ContactOut People Search Integration

### Overview
Search and import matching candidate profiles from ContactOut's extensive database of verified professionals.

### Key Features

#### 🔍 Smart Skill Matching
- Automatically matches candidates based on job requirements with percentage scoring
- Advanced algorithm analyzes job descriptions and candidate profiles
- Provides compatibility scores to help identify the best matches

#### 📊 API Usage Statistics (NEW)
Real-time monitoring of ContactOut quota consumption with comprehensive tracking:

- **Profile Access Tracking**: Monitor candidate profile views and downloads
- **Phone Access Monitoring**: Track phone number access usage
- **Search Query Counting**: Monitor search API calls and remaining quotas
- **Color-Coded Indicators**: 
  - 🟢 Green (0-50% usage)
  - 🔵 Blue (50-75% usage)
  - 🟡 Yellow (75-90% usage)
  - 🔴 Red (90%+ usage)
- **Billing Period Display**: Shows current monthly billing cycle and renewal dates
- **Smart Alerts**: Critical usage warnings and quota management notifications
- **Progress Bars**: Visual representation of quota consumption across all service types

#### 🔐 License-based Access
- Secure integration requiring valid ContactOut API key from ContactOut directly
- Authentication ensures authorized access to candidate database
- API key validation and secure storage

#### 📥 Bulk Candidate Import
- Select multiple candidates and import them directly for resume scoring
- Streamlined workflow from search to import
- Batch processing capabilities for efficient recruitment

#### ⭐ Top Match Highlighting
- Best matching candidates are highlighted and sorted to the top
- Priority ranking based on skill compatibility
- Visual indicators for top-tier matches

#### 📋 Interactive Setup Guide
- Step-by-step guidance for API key configuration and usage
- Built-in tutorials and help documentation
- Error handling and troubleshooting support

## 🔍 Similar JD Search (NEW)

### Overview
Search matching profiles from existing resumes from other similar job descriptions in your database.

### Key Features

#### 🎯 Skill-Based Matching
- Find candidates from similar JDs based on skill overlap and compatibility
- Analyze skill sets across different job postings
- Identify transferable skills and relevant experience

#### 📊 Smart JD Ranking
- Similar job descriptions ranked by skill match percentage
- Intelligent algorithm compares job requirements
- Prioritizes most relevant historical job postings

#### 🔄 Cross-JD Profile Search
- Search through qualified candidates from multiple previous job postings
- Leverage your existing candidate database
- Discover hidden talent from past recruitment cycles

#### 📤 Bulk Profile Import
- Select and import top matching profiles from historical data
- Efficient batch processing of candidate data
- Streamlined workflow for candidate re-engagement

#### 📋 Comprehensive Candidate Data
- Access to previous application data, scores, and qualification status
- Historical performance metrics and assessment results
- Complete candidate journey tracking

## Getting Started

### Prerequisites
1. **ContactOut Integration**: Valid ContactOut API key and subscription
2. **Historical Data**: Existing job descriptions and candidate profiles for Similar JD Search
3. **System Access**: Appropriate permissions for Resume Pipeline Connectors

### Setup Process
1. Navigate to **Resume Parser** → **Resume Pipeline Connectors**
2. Configure your desired connector (ContactOut or Similar JD Search)
3. Follow the interactive setup guide for API configuration
4. Begin searching and importing candidates

### Best Practices
- Monitor API usage regularly to avoid quota overages
- Use skill-based matching to improve candidate quality
- Leverage historical data for faster candidate sourcing
- Set up smart alerts for quota management
- Review and validate imported candidate data
