# Features for small businesses

This section provides a detailed explanation of the assessment and recruitment automation workflows involved in Starter and Pro plans perfect for teams and businesses.

## Turiyaskills  Starter and Pro Plans:

Before starting the assessment creation process, the following steps are completed:

1. **Accessing the App**

   - The super user (admin of the organization) logs into a App provided for the Pro plan purchase.
   - The website URL includes the App's domain name, e.g., `app.turiyaskills.co`, where "turiyaskills" represents the App's name.

2. **Activating the Pro Plan**

   - Upon logging in, a new plan called "Pro" is activated for the organization.
   - This package adds credits to the organization's wallet, which includes:
     - **AI Credits:** For AI-driven features such as automated assessment creation.
     - **Response Credits:** For processing candidate responses.

3. **Utilizing Credits Across the Organization**

   - Users with the same domain email address (e.g., `@organization.com`) can be invited by Org Admin, signup, log into the App and access these credits to use SMARTEVAL (Quiz), ASSESS360 (Survey).
   - Any usage of credits by users within the organization's domain, including the super user, deducts from the shared wallet balance.
   - Billing Info page will display Current plan section with Usage History button to show each individuals Credit usage records as its now in Pro Plan
   - Billing Info page for Business Org Admin shall contain Current plan with individual credit usage history, Team info with entire team's plan detail and credit usage history 

4. **Team Management Feature**

   - The super user can use the setting menu>Team to do team management
   - The super user can Invite user to add to his/her team, make a user an admin and also remove user.
   - This allows them to review activities, monitor credit usage, and manage processes effectively.

With the Pro plan in place, the organization is equipped to proceed with the assessment workflow seamlessly.

## Recruitment Automation Workflow - Pro Plan


This plan will have both Turiyaskills App Assessment workflow and Top1 App Recruitment Automation Workflow

1. **Job Description (JD) Upload**

   - The process begins with uploading a Job Description (JD) containing essential details, including:
     - Role requirements.
     - Responsibilities.
     - Candidate qualifications.
   - The JD forms the foundation for sourcing, filtering, and evaluation processes.
   - Ability to create Assessments automatically from uploaded, parsed JD

2. **Resume Collection**

   - Resumes are sourced from multiple platforms based on JD requirements, such as:
     - Job Portals
     - Professional Networking Platforms
     - Website Job Board
     - and Uploaded into Resume Parser
   - All collected resumes are parsed, scored based on the experience, education, & skills of the candidate against the requirement of the JD.

3. **Resume Processing (Parsing and Evaluation)**

   - The system processes resumes by:
     - Extracting key information (skills, qualifications, work experience).
     - Comparing the data against JD criteria.
     - Storing relevant information in the database.

4. **Filtering Resumes**

   - Resumes are filtered to shortlist candidates:
     - **Rejected/Duplicated:** Resumes that don’t meet criteria are excluded.
     - **Selected:** Resumes meeting criteria move to the next stage for assessment taggig and publish to emails.

## Assessment Creation and Execution

5. **Assessment Creation**

   - Assessments are designed to evaluate shortlisted candidates based on the job role:
     - **Manual:** Recruiters create tailored assessments.
     - **AI-Based:** Automated systems generate assessments using JD and candidate data.
   - Assessments are finalized and published on the platform for candidates.

6. **Sending Assessment Links**

   - The system generates and shares assessment links with candidates via email.
   - Candidates complete the assessments within a specified timeframe (if applicable).

7. **Evaluation and Advancement**

   - **Score Evaluation:**
     - Responses are reviewed and scores calculated.
     - Scores are compared against benchmarks.
   - **Shortlisting:**
     - Candidates meeting score criteria are shortlisted.
   - **Advancement:**
     - Shortlisted candidates proceed to the next phase.
