# Enhancing Team Recruitment

Turiyaskills is excited to announce the addition of several team-specific features designed to significantly improve the recruitment process:

## 🏢 Centralized JD & Resume Management

This new feature provides a single, consolidated location for all Job Descriptions (JDs) and candidate resumes. It offers transparency by allowing team members to see which JDs and resumes are currently under review and by whom, thereby boosting collaboration and overall efficiency.

### Key Benefits:
- **Single Source of Truth**: All job descriptions and resumes in one centralized location
- **Real-time Visibility**: Track who is reviewing which documents
- **Enhanced Collaboration**: Team members can see current review status and workload distribution
- **Improved Efficiency**: Eliminates confusion about document ownership and review progress

## 📋 Common Assessment Sharing

To ensure consistency and reduce redundant efforts, this feature enables teams to utilize a standardized assessment for candidates. This ensures that all team members are aligned in their evaluation criteria, leading to a more streamlined and fair assessment process.

### Key Benefits:
- **Standardized Evaluation**: Consistent assessment criteria across all team members
- **Reduced Redundancy**: Eliminate duplicate assessment creation efforts
- **Fair Assessment Process**: Ensures all candidates are evaluated using the same standards
- **Team Alignment**: All team members work with unified evaluation frameworks

## 🔄 Ownership Transfer of Artefacts

This feature allows team members to transfer ownership of artefacts (e.g., assessments, JDs) they have created or uploaded to other team members. This ensures smooth task transitions and efficient task completion.

### Key Benefits:
- **Seamless Handovers**: Easy transfer of work between team members
- **Flexible Task Management**: Reassign responsibilities based on availability and expertise
- **Continuity Assurance**: Maintain project momentum during team changes
- **Efficient Task Completion**: Optimize resource allocation and workload distribution

## 🎯 Impact on Recruitment Workflow

These new features are specifically aimed at:

- **Streamlining the recruitment workflow**
- **Improving communication among team members**
- **Increasing the overall efficiency of candidate evaluations**
- **Enhancing team collaboration and transparency**
- **Reducing administrative overhead**

## Getting Started

1. **Access Team Features**: Navigate to your team dashboard to access centralized management tools
2. **Setup Permissions**: Configure team member access levels and permissions
3. **Share Assessments**: Begin utilizing common assessment templates for consistent evaluation
4. **Transfer Ownership**: Use the ownership transfer feature to reassign tasks as needed

These features are now available to all team accounts and can be accessed through the main team management interface.
