# How to Install Safe Exam Browser and Access Assessment

This guide provides a step-by-step process for installing the Safe Exam Browser (SEB) and accessing assessments on the Turiyaskills platform. It simplifies the setup, ensuring users can securely complete their online assessments without confusion. By following the instructions, users can efficiently navigate the installation and verification process, enhancing their exam experience. Viewing this guide is essential for anyone looking to ensure a smooth and secure assessment process.

## Overview

Safe Exam Browser (SEB) is a secure testing environment that prevents unauthorized access to other applications and websites during assessments. This ensures exam integrity and provides a controlled testing environment for candidates.

## Important Notes

### 🔒 Security Features
- **Browser Restrictions**: SEB prevents access to other applications and websites during the exam
- **Controlled Environment**: Ensures exam integrity through secure testing conditions
- **Photo Capture**: May include candidate photo verification during the assessment

### 📋 Requirements
- **Compatible Device**: Windows or Mac computer with SEB installed
- **Stable Internet**: Reliable internet connection for assessment completion
- **Email Access**: Active email for OTP verification and instructions

### ⚠️ Troubleshooting Tips
- **Installation Issues**: Ensure you download the correct version for your operating system
- **OTP Problems**: Check spam/junk folders if OTP email is delayed
- **SEB Configuration**: Make sure to download and use the specific .SEB file provided
- **Technical Support**: Contact platform support if you encounter installation difficulties

## Best Practices

1. **Pre-Installation**: Install Safe Exam Browser before the scheduled assessment time
2. **Test Run**: Perform a test run to ensure SEB is working correctly
3. **Backup Plan**: Have technical support contact information readily available
4. **Environment Setup**: Ensure a quiet, private space for the assessment
5. **System Check**: Verify camera and microphone functionality if required

For additional support with Safe Exam Browser setup, please contact our technical support team or refer to the [Assessment App Features](/features/assessment-platform) documentation.
